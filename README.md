# Digital Detectives (Static Classroom Activity)

## Overview
This project is a fictional middle school digital literacy investigation. Students analyze multiple online-style sources to determine whether a viral school-cancellation claim is trustworthy.

## Included Pages
- `index.html` – Case file and instructions
- `social.html` – Fictional social profile
- `messages.html` – Fictional text thread
- `email.html` – Fictional inbox
- `search.html` – Fictional search results
- `evidence.html` – Notes worksheet with localStorage + print
- `teacher-notes.html` – Answer key, clues, discussion prompts, and rubric

## Safety and Content Notes
- All names, schools, usernames, phone/email-style data, and locations are fictional.
- No real social media brand names or branding are used.
- Content is school-appropriate for grade 8.

## Setup / Run
1. Download or clone the folder.
2. Open `index.html` in a web browser.
3. Navigate through evidence pages and complete `evidence.html`.

No backend, database, or build tools are required.
